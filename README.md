## 💳명함 관리 프로그램 사용법
1. 로그인 페이지에 입장한다.
2. 회원가입 페이지에 입장하여, 계정 생성을 한다.
3. 메인 리스트 화면에 들어간다.
4. 명함 등록 버튼을 클릭하여 명함 등록 페이지로 이동한다.
5. 명함 생성을 한다.
6. 명함을 클릭하면 명함에 대한 상세 내역이 나오게 된다.

## ✅ 구현 현황
1. 명함 CREATE, READ, UPDATE, DELETE
2. LOGIN, LOGOUT
3. MOBILE/WEB 반응형 구현완료
